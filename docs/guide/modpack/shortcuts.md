::: tip

- 整合包中内置了一个“可视化键位图”可通过`选项→按键控制→可视化键位图`打开
- 如果按键与其他模组冲突，可自行调整
:::

> 下方表格只举例被修改过，冷门模组的按键
>
> 下表中“未指定”表示该功能未绑定任何键位（即原键位已被清除）。

## 冷门模组

| 模组 | 功能 | 键位 |
| --- | --- | --- |
| Toggle Nametags | 隐藏玩家头上的名牌（Toggle Nametags） | `-` |
| - | Toggle Boss Bar | `]` → 未指定 |
| - | Show your own nametag | `M` → 未指定 |
| 更好的 Tab 列表 | 切换模组 | 未指定 |
| - | 向右滚动 | 右方向键 |
| - | 向左滚动 | 左方向键 |
| - | 打开配置 | 未指定 |
| ClickThrough Plus | Turn ClickThrough Plus on/off | `F9` → 未指定 |
