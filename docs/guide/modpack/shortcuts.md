:::

> 下方表格只举例被修改过，冷门模组的按键
>
> 键位不分前后

## 冷门模组

| 模组 | 功能 | 键位 |
| --- | --- | --- |
| Toggle Nametags | 隐藏玩家头上的名牌（Toggle Nametags） | `-` |
| | Toggle Boss Bar | 未指定 |
| | Show your own nametag | 未指定 |
